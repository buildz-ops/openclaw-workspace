# chat.findSpaceByName

**Server:** google-workspace

## Description
Finds a Google Chat space by its display name.

## Parameters
```json
{
  "displayName": {
    "type": "string",
    "description": "The display name of the space to find."
  }
}
```

**Required:** displayName

## Usage
```bash
mcporter call google-workspace.chat.findSpaceByName [args...]
```
