# slides.getMetadata

**Server:** google-workspace

## Description
Gets metadata about a Google Slides presentation.

## Parameters
```json
{
  "presentationId": {
    "type": "string",
    "description": "The ID or URL of the presentation."
  }
}
```

**Required:** presentationId

## Usage
```bash
mcporter call google-workspace.slides.getMetadata [args...]
```
