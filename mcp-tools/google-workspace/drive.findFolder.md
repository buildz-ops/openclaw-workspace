# drive.findFolder

**Server:** google-workspace

## Description
Finds a folder by name in Google Drive.

## Parameters
```json
{
  "folderName": {
    "type": "string",
    "description": "The name of the folder to find."
  }
}
```

**Required:** folderName

## Usage
```bash
mcporter call google-workspace.drive.findFolder [args...]
```
