# sheets.getMetadata

**Server:** google-workspace

## Description
Gets metadata about a Google Sheets spreadsheet.

## Parameters
```json
{
  "spreadsheetId": {
    "type": "string",
    "description": "The ID or URL of the spreadsheet."
  }
}
```

**Required:** spreadsheetId

## Usage
```bash
mcporter call google-workspace.sheets.getMetadata [args...]
```
