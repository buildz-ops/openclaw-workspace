# docs.extractIdFromUrl

**Server:** google-workspace

## Description
Extracts the document ID from a Google Workspace URL.

## Parameters
```json
{
  "url": {
    "type": "string",
    "description": "The URL of the Google Workspace document."
  }
}
```

**Required:** url

## Usage
```bash
mcporter call google-workspace.docs.extractIdFromUrl [args...]
```
